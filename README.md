## 📌 Overview

A **fully responsive personal portfolio website** built as my first task for the **M Tech Web Internship**. This website serves as my professional online presence, showcasing my skills, experience, projects, and contact information.

> **Task:** CV/Portfolio Website  
> **Internship:** M Tech Production & Marketing  
> **Role:** Full Stack Web Developer (Intern)

---

## ✨ Features

### 🏠 **Home / Hero Section**
- Professional profile photo
- Personal introduction with animated typing effect
- Quick navigation pills (Resume, Projects, Contact)

### 📄 **Resume Section**
- **Work Experience** – 5+ positions with dates and descriptions
- **Education** – Software Engineering, ICS, Matriculation
- **Skills & Expertise** – Graphic Designing, Front-End, Backend, Tools, Programming

### 🚀 **Projects Section**
- Project showcase with descriptions
- Placeholder for upcoming projects

### 📬 **Contact Section**
- Contact form with **client-side validation**
- Email, location, and LinkedIn integration

### 🎨 **Interactive Elements**
- Mobile-responsive navigation (hamburger menu)
- Active section highlighting on scroll
- Back-to-top button
- Smooth scrolling animations
- Hover effects on buttons and cards

### 📱 **Fully Responsive**
- Optimized for **Desktop**, **Tablet**, and **Mobile** devices
- Breakpoints at 900px and 600px
